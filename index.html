Understanding exit codes on Linux

Whenever you run a command in a Linux terminal, a numeric exit code is generated – even if you see no sign of it. The code reports the success or failure of the command. And, if the command fails, it provides a value that hints at the problem. If the exit code is 0, all is well. (Network World)
